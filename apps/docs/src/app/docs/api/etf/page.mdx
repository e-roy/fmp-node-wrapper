# ETF Endpoints

Access ETF (Exchange-Traded Fund) data including quotes, profiles, and holdings information.

## Overview

The ETF endpoints provide access to ETF market data, including real-time quotes, fund profiles, holdings information, and ETF listings.

## Quick Start

<CodeBlock language="typescript">
{`import { FMP } from 'fmp-node-api';

const fmp = new FMP({ apiKey: 'your-api-key' });

// Get ETF quote
const quote = await fmp.etf.getQuote({ symbol: 'SPY' });

// Get ETF profile
const profile = await fmp.etf.getProfile({ symbol: 'SPY' });

// Get ETF holdings
const holdings = await fmp.etf.getHoldings({ symbol: 'SPY' });`}

</CodeBlock>

## Available Methods

### getQuote

Get real-time ETF quote.

<ApiTable
  endpoints={[
    {
      method: 'GET',
      path: '/quote/{symbol}',
      description: 'Get real-time ETF quote',
    },
  ]}
/>

**Parameters:**

| Parameter | Type     | Required | Description              |
| --------- | -------- | -------- | ------------------------ |
| `symbol`  | `string` | Yes      | ETF symbol (e.g., 'SPY') |

**Example:**

<CodeBlock language="typescript">
{`const quote = await fmp.etf.getQuote({ symbol: 'SPY' });

if (quote.success) {
console.log('Price:', quote.data[0].price);
console.log('Change:', quote.data[0].change);
console.log('Volume:', quote.data[0].volume);
}`}

</CodeBlock>

<ApiTable
  endpoints={[

    },
  ]}

**Parameters:**

| Parameter | Type     | Required | Description              |

### getHoldings

Get ETF holdings and composition data.

<ApiTable
  endpoints={[
    {
      method: 'GET',
      path: '/etf-holder/{symbol}',
      description: 'Get ETF holdings',
    },
  ]}
/>

**Parameters:**

| Parameter | Type     | Required | Description                  |
| --------- | -------- | -------- | ---------------------------- |
| `symbol`  | `string` | Yes      | ETF symbol (e.g., 'SPY')     |
| `limit`   | `number` | No       | Number of holdings to return |
| `page`    | `number` | No       | Page number for pagination   |

**Example:**

<CodeBlock language="typescript">
{`const holdings = await fmp.etf.getHoldings({ 
    symbol: 'SPY',
    limit: 10 
});

if (holdings.success) {
console.log('Holdings:', holdings.data);
}`}

</CodeBlock>

## ETF Categories

ETFs can be categorized by their investment focus:

- **Equity ETFs** - Track stock indices or sectors
- **Bond ETFs** - Track bond indices
- **Commodity ETFs** - Track commodity prices
- **International ETFs** - Track foreign markets
- **Sector ETFs** - Track specific industry sectors

## Rate Limiting

ETF endpoints are subject to FMP's rate limits. For production applications, consider implementing appropriate rate limiting strategies.
