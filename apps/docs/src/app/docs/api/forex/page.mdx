# Forex Endpoints

Access foreign exchange data including real-time quotes, historical data, and currency information.

## Overview

The forex endpoints provide access to foreign exchange market data, including real-time currency quotes, historical performance data, and currency pair information.

## Quick Start

<CodeBlock language="typescript">
{`import { FMP } from 'fmp-node-api';

const fmp = new FMP({ apiKey: 'your-api-key' });

// Get forex quote
const quote = await fmp.forex.getQuote({ symbol: 'EURUSD' });

// Get historical forex data
const historical = await fmp.forex.getHistoricalPrice({
    symbol: 'EURUSD',
    from: '2024-01-01',
    to: '2024-01-31'
});`}
</CodeBlock>

## Available Methods

### getQuote

Get real-time forex quote for a currency pair.

<ApiTable
  endpoints={[
    {
      method: 'GET',
      path: '/quote/{symbol}',
      description: 'Get real-time forex quote',
    },
  ]}
/>

**Parameters:**

| Parameter | Type     | Required | Description                    |
| --------- | -------- | -------- | ------------------------------ |
| `symbol`  | `string` | Yes      | Currency pair symbol (e.g., 'EURUSD') |

**Example:**

<CodeBlock language="typescript">
{`const quote = await fmp.forex.getQuote({ symbol: 'EURUSD' });

if (quote.success) {
    console.log('Exchange Rate:', quote.data[0].price);
    console.log('Change:', quote.data[0].change);
    console.log('Change %:', quote.data[0].changesPercentage);
}`}
</CodeBlock>

### getHistoricalPrice

Get historical forex data for a currency pair.

<ApiTable
  endpoints={[
    {
      method: 'GET',
      path: '/historical-price-full/{symbol}',
      description: 'Get historical forex data',
    },
  ]}
/>

**Parameters:**

| Parameter     | Type     | Required | Description                    |
| ------------- | -------- | -------- | ------------------------------ |
| `symbol`      | `string` | Yes      | Currency pair symbol (e.g., 'EURUSD') |
| `from`        | `string` | No       | Start date (YYYY-MM-DD)       |
| `to`          | `string` | No       | End date (YYYY-MM-DD)         |
| `timeseries`  | `number` | No       | Number of days for timeseries |

**Example:**

<CodeBlock language="typescript">
{`const historical = await fmp.forex.getHistoricalPrice({
    symbol: 'EURUSD',
    from: '2024-01-01',
    to: '2024-01-31'
});

if (historical.success) {
    console.log('Historical Data:', historical.data.historical);
}`}
</CodeBlock>

### getForexList

Get list of all available forex pairs.

<ApiTable
  endpoints={[
    {
      method: 'GET',
      path: '/forex',
      description: 'Get list of all forex pairs',
    },
  ]}
/>

**Example:**

<CodeBlock language="typescript">
{`const forexList = await fmp.forex.getForexList();

if (forexList.success) {
    console.log('Available Pairs:', forexList.data);
}`}
</CodeBlock>

## Common Currency Pairs

Here are some common currency pairs you can use:

- **EURUSD** - Euro / US Dollar
- **GBPUSD** - British Pound / US Dollar
- **USDJPY** - US Dollar / Japanese Yen
- **USDCHF** - US Dollar / Swiss Franc
- **AUDUSD** - Australian Dollar / US Dollar
- **USDCAD** - US Dollar / Canadian Dollar
- **NZDUSD** - New Zealand Dollar / US Dollar

## Error Handling

<CodeBlock language="typescript">
{`const response = await fmp.forex.getQuote({ symbol: 'INVALID' });

if (response.success) {
    // Handle successful response
    console.log(response.data);
} else {
    // Handle error
    console.error('Error:', response.error);
    console.error('Status:', response.status);
}`}
</CodeBlock>
