# Calendar Endpoints

The Calendar Endpoints provide access to various financial calendars including earnings, dividends, economic events, IPOs, and stock splits. These endpoints help you track important financial events and announcements.

## Get Earnings Calendar

Retrieve upcoming earnings announcements with estimated and actual earnings per share (EPS) and revenue data.

<CodeBlock language="typescript">
  {`const earningsCalendar = await fmp.calendar.getEarningsCalendar({
    from: '2024-01-01',
    to: '2024-01-31'
});`}
</CodeBlock>

## Working with Calendar Data

### Filtering by Date Range

All calendar endpoints support optional date filtering. When no dates are provided, the API returns the most recent data:

<CodeBlock language="typescript">
{`// Get all upcoming earnings (no date filter)
const allEarnings = await fmp.calendar.getEarningsCalendar({});

// Get earnings for specific month
const janEarnings = await fmp.calendar.getEarningsCalendar({
from: '2024-01-01',
to: '2024-01-31'
});

// Get earnings for next quarter
const nextQuarter = await fmp.calendar.getEarningsCalendar({
from: '2024-01-01',
to: '2024-03-31'
});`}

</CodeBlock>

### Processing Calendar Events

Calendar data is particularly useful for building financial dashboards and tracking important events:

<CodeBlock language="typescript">
{`// Get all calendar events for a specific date
const targetDate = '2024-01-25';

const [earnings, earningsConfirmed, dividends, economic, ipos, splits] = await Promise.all([
fmp.calendar.getEarningsCalendar({ from: targetDate, to: targetDate }),
fmp.calendar.getEarningsConfirmed({ from: targetDate, to: targetDate }),
fmp.calendar.getDividendsCalendar({ from: targetDate, to: targetDate }),
fmp.calendar.getEconomicsCalendar({ from: targetDate, to: targetDate }),
fmp.calendar.getIPOCalendar({ from: targetDate, to: targetDate }),
fmp.calendar.getSplitsCalendar({ from: targetDate, to: targetDate })
]);

console.log('Earnings:', earnings.data.length);
console.log('Earnings Confirmed:', earningsConfirmed.data.length);
console.log('Dividends:', dividends.data.length);
console.log('Economic Events:', economic.data.length);
console.log('IPOs:', ipos.data.length);
console.log('Splits:', splits.data.length);`}

</CodeBlock>

## Error Handling

Always check the success property before accessing data:

<CodeBlock language="typescript">
{`const earnings = await fmp.calendar.getEarningsCalendar({
    from: '2024-01-01',
    to: '2024-01-31'
});

if (earnings.success) {
console.log('Earnings events:', earnings.data.length);
earnings.data.forEach(event => {
console.log(\`\${event.symbol}: \${event.estimatedEps} EPS\`);
});
} else {
console.error('Error:', earnings.error);
console.error('Status:', earnings.status);
}`}

</CodeBlock>

## Rate Limiting

Calendar endpoints are subject to FMP's rate limits. For production applications, implement appropriate rate limiting and caching strategies, especially when fetching data for extended date ranges.
