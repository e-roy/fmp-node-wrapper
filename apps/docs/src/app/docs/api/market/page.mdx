### MarketHours

<CodeBlock language="typescript">
{`interface MarketHours {
  stockExchangeName: string;
  stockMarketHours: {
    openingHour: string;
    closingHour: string;
  };
  stockMarketHolidays: MarketHoliday[];
  isTheStockMarketOpen: boolean;
  isTheEuronextMarketOpen: boolean;
  isTheForexMarketOpen: boolean;
  isTheCryptoMarketOpen: boolean;
}

interface MarketHoliday {
year: number;
'Martin Luther King, Jr. Day': string;
"Presidents' Day": string;
'Good Friday': string;
'Memorial Day': string;
Juneteenth: string;
'Independence Day': string;
'Labor Day': string;
'Thanksgiving Day': string;
Christmas: string;
}`}

</CodeBlock>

### MarketSectorPerformance

<CodeBlock language="typescript">
  {`interface MarketSectorPerformance {

## Error Handling

Always check the success property before accessing data:

<CodeBlock language="typescript">
{`const performance = await fmp.market.getMarketPerformance();

if (performance.success && performance.data) {
console.log('Market Performance:', performance.data);
} else {
console.error('Error:', performance.error);
console.error('Status:', performance.status);
}`}

</CodeBlock>

## Rate Limiting

Market endpoints are subject to FMP's rate limits. For production applications, implement appropriate rate limiting and caching strategies.

## Market Hours

Market data availability depends on trading hours:

- **Stock Market** - Typically 9:30 AM - 4:00 PM ET (Monday-Friday)
- **Pre-Market** - 4:00 AM - 9:30 AM ET
- **After-Hours** - 4:00 PM - 8:00 PM ET
- **Weekends/Holidays** - Limited data availability
