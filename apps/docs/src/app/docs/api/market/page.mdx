# Market Endpoints

Access market-wide data including performance metrics, trading hours, and sector information.

## Overview

The market endpoints provide access to broad market data, including overall market performance, trading hours, sector performance, and market indices.

## Quick Start

<CodeBlock language="typescript">
{`import { FMP } from 'fmp-node-api';

const fmp = new FMP({ apiKey: 'your-api-key' });

// Get market performance
const performance = await fmp.market.getMarketPerformance();

// Get market hours
const hours = await fmp.market.getMarketHours();

// Get sector performance
const sectors = await fmp.market.getSectorPerformance();`}
</CodeBlock>

### getMarketIndex

Get market index data with optional date range.

<ApiTable
  endpoints={[
    {

      path: '/market-index',
      description: 'Get market index data',
    },
  ]}

**Parameters:**

| Parameter | Type     | Required | Description                    |
| --------- | -------- | -------- | ------------------------------ |
| `from`    | `string` | No       | Start date (YYYY-MM-DD)       |
| `to`      | `string` | No       | End date (YYYY-MM-DD)         |

**Example:**

<CodeBlock language="typescript">
{`const index = await fmp.market.getMarketIndex({
  from: '2024-01-01',

  console.log('Market Index:', index.data);
}`}
</CodeBlock>

## Market Data Insights

Market endpoints provide valuable insights for:

- **Market Sentiment** - Understanding overall market direction
- **Sector Rotation** - Identifying which sectors are performing well
- **Trading Activity** - Monitoring volume and activity levels
- **Market Timing** - Understanding when markets are open/closed

## Common Use Cases

### Market Dashboard

<CodeBlock language="typescript">
{`// Create a market dashboard
const [performance, sectors, hours] = await Promise.all([
  fmp.market.getMarketPerformance(),
  fmp.market.getSectorPerformance(),
  fmp.market.getMarketHours()
]);

console.log('Market Status:', hours.data);
console.log('Top Performers:', performance.data.slice(0, 5));
console.log('Sector Leaders:', sectors.data.slice(0, 3));`}
</CodeBlock>

### Market Analysis

<CodeBlock language="typescript">
{`// Analyze market breadth
const [gainers, losers] = await Promise.all([
  fmp.market.getGainers(),
  fmp.market.getLosers()
]);

const advancing = gainers.data.length;
const declining = losers.data.length;
const breadth = advancing - declining;

console.log('Market Breadth:', breadth);`}
</CodeBlock>

## Rate Limiting

Market endpoints are subject to FMP's rate limits. For production applications, consider implementing appropriate rate limiting strategies.

## Market Hours

Market data availability depends on trading hours:

- **Stock Market** - Typically 9:30 AM - 4:00 PM ET (Monday-Friday)
- **Pre-Market** - 4:00 AM - 9:30 AM ET
- **After-Hours** - 4:00 PM - 8:00 PM ET
- **Weekends/Holidays** - Limited data availability
