# Senate & House Trading Endpoints

Access congressional trading data including senate and house member transactions through both direct API calls and RSS feeds.

## Overview

The Senate & House Trading endpoints provide access to congressional trading data, allowing you to track stock transactions made by members of Congress. This data is valuable for understanding potential conflicts of interest and market sentiment based on political insider trading.

## Quick Start

<CodeBlock language="typescript">
{`import { FMP } from 'fmp-node-api';

const fmp = new FMP({ apiKey: 'your-api-key' });

// Get senate trading data for Apple
const senateTrading = await fmp.senateHouse.getSenateTrading({
symbol: 'AAPL'
});

// Get house trading data for Microsoft
const houseTrading = await fmp.senateHouse.getHouseTrading({
symbol: 'MSFT'
});

// Get senate trading RSS feed
const senateRSS = await fmp.senateHouse.getSenateTradingRSSFeed({
page: 0
});

// Get house trading RSS feed
const houseRSS = await fmp.senateHouse.getHouseTradingRSSFeed({
page: 0
});`}

</CodeBlock>

## Use Cases

### 1. Track Political Trading by Stock

Monitor which politicians are trading specific stocks.

<CodeBlock language="typescript">
  {`// Track Apple trading by politicians
const appleTrading = await fmp.senateHouse.getSenateTrading({
  symbol: 'AAPL'
});

console.log('Senate members trading Apple:', appleTrading.data);

const houseAppleTrading = await fmp.senateHouse.getHouseTrading({
symbol: 'AAPL'
});

console.log('House members trading Apple:', houseAppleTrading.data);`}

</CodeBlock>

### 2. Monitor Recent Trading Activity

Use RSS feeds to get the latest trading disclosures.

<CodeBlock language="typescript">
  {`// Get latest senate trading activity
const latestSenateTrading = await fmp.senateHouse.getSenateTradingRSSFeed({
  page: 0
});

console.log('Latest senate trading:', latestSenateTrading.data);

// Get latest house trading activity
const latestHouseTrading = await fmp.senateHouse.getHouseTradingRSSFeed({
page: 0
});

console.log('Latest house trading:', latestHouseTrading.data);`}

</CodeBlock>

### 3. Analyze Trading Patterns

Compare trading activity across different stocks and chambers.

<CodeBlock language="typescript">
  {`// Compare senate vs house trading for tech stocks
const techStocks = ['AAPL', 'MSFT', 'GOOGL', 'TSLA'];

for (const symbol of techStocks) {
const senateTrading = await fmp.senateHouse.getSenateTrading({ symbol });
const houseTrading = await fmp.senateHouse.getHouseTrading({ symbol });

console.log(symbol + ':');
console.log(' Senate transactions: ' + (senateTrading.data?.length || 0));
console.log(' House transactions: ' + (houseTrading.data?.length || 0));
}`}

</CodeBlock>

### 4. Build Trading Alerts

Create alerts for when politicians trade specific stocks.

<CodeBlock language="typescript">
  {`// Check for recent trading in a specific stock
async function checkRecentTrading(symbol: string) {
  const senateTrading = await fmp.senateHouse.getSenateTrading({ symbol });
  const houseTrading = await fmp.senateHouse.getHouseTrading({ symbol });
  
  const allTrading = [
    ...(senateTrading.data || []),
    ...(houseTrading.data || [])
  ];
  
  // Filter for recent transactions (last 30 days)
  const thirtyDaysAgo = new Date();
  thirtyDaysAgo.setDate(thirtyDaysAgo.getDate() - 30);
  
  const recentTrading = allTrading.filter(trade => {
    const tradeDate = new Date(trade.transactionDate);
    return tradeDate >= thirtyDaysAgo;
  });
  
  if (recentTrading.length > 0) {
    console.log('Alert: ' + recentTrading.length + ' recent trades in ' + symbol);
recentTrading.forEach(trade => {
  console.log('  ' + trade.owner + ' ' + trade.type + ' ' + trade.amount);
});
  }
}

// Check for Apple trading
await checkRecentTrading('AAPL');`}

</CodeBlock>

## Pagination

Both RSS feed endpoints support pagination to access historical data:

<CodeBlock language="typescript">
  {`// Get multiple pages of senate trading data
const pages = [0, 1, 2, 3, 4];
const allSenateTrading = [];

for (const page of pages) {
const result = await fmp.senateHouse.getSenateTradingRSSFeed({ page });
if (result.success && result.data) {
allSenateTrading.push(...result.data);
}
}

console.log('Total senate trading records: ' + allSenateTrading.length);`}

</CodeBlock>

## Rate Limits

The senate-house trading endpoints respect FMP's rate limits. For applications requiring frequent updates, consider implementing appropriate caching strategies.

## Data Availability

- Trading data is typically available within 45 days of transaction disclosure
- Historical data may be limited based on your subscription plan
- RSS feeds provide the most recent disclosures first
- Direct symbol queries return all available data for that specific stock

## Error Handling

Always check the success property before accessing data:

<CodeBlock language="typescript">
{`const result = await fmp.senateHouse.getSenateTrading({ symbol: 'AAPL' });

if (result.success) {
console.log('Senate trading data:', result.data);
} else {
console.error('Error:', result.error);
console.error('Status:', result.status);
}`}

</CodeBlock>
