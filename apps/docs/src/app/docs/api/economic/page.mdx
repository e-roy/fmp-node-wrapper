# Economic Endpoints

Access economic indicators, calendar data, and treasury rates.

## Overview

The economic endpoints provide access to economic data, including economic indicators, calendar events, and treasury rates that can impact financial markets.

## Quick Start

<CodeBlock language="typescript">
{`import { FMP } from 'fmp-node-api';

const fmp = new FMP({ apiKey: 'your-api-key' });

// Get economic calendar
const calendar = await fmp.economic.getEconomicCalendar();

// Get treasury rates
const rates = await fmp.economic.getTreasuryRates();

// Get economic indicators
const indicators = await fmp.economic.getEconomicIndicators();`}

</CodeBlock>

## Key Economic Indicators

Economic data includes important indicators such as:

- **GDP** - Gross Domestic Product growth
- **Inflation** - Consumer Price Index (CPI)
- **Unemployment** - Unemployment rate
- **Interest Rates** - Federal Reserve rates
- **Treasury Yields** - Government bond yields
- **Consumer Confidence** - Economic sentiment

## Treasury Yield Curve

The treasury yield curve is a key economic indicator:

<CodeBlock language="typescript">
{`// Analyze yield curve
const rates = await fmp.economic.getTreasuryRates();

if (rates.success) {
const current = rates.data[0];
const shortTerm = current.month3; // 3-month rate
const longTerm = current.year10; // 10-year rate

const yieldCurve = longTerm - shortTerm;
console.log('Yield Curve Spread:', yieldCurve);

if (yieldCurve < 0) {
console.log('Inverted Yield Curve - Recession indicator');
}
}`}

</CodeBlock>

## Economic Calendar Analysis

<CodeBlock language="typescript">
{`// Filter high-impact economic events
const calendar = await fmp.economic.getEconomicCalendar();

if (calendar.success) {
const highImpact = calendar.data.filter(event =>

console.log('High Impact Events:', highImpact);
}`}

</CodeBlock>

## Common Use Cases

### Economic Dashboard

<CodeBlock language="typescript">
{`// Create economic dashboard
const [calendar, rates, indicators] = await Promise.all([
  fmp.economic.getEconomicCalendar(),
  fmp.economic.getTreasuryRates(),
  fmp.economic.getEconomicIndicators()
]);

console.log('Upcoming Events:', calendar.data.slice(0, 5));
console.log('Current Rates:', rates.data[0]);
console.log('Key Indicators:', indicators.data);`}

</CodeBlock>

### Market Impact Analysis

<CodeBlock language="typescript">
{`// Analyze economic impact on markets
const [rates, inflation] = await Promise.all([
  fmp.economic.getTreasuryRates(),
  fmp.economic.getInflationData()
]);

const realRate = rates.data[0].year10 - inflation.data[0].value;
console.log('Real Interest Rate:', realRate);`}

</CodeBlock>

## Rate Limiting

Economic endpoints are subject to FMP's rate limits. For production applications, consider implementing appropriate rate limiting strategies.

## Data Frequency

Economic data is typically updated:

- **Real-time** - Treasury rates, some indicators
- **Daily** - Economic calendar
- **Monthly** - Most economic indicators
- **Quarterly** - GDP data
