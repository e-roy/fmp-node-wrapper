# Company Endpoints

The Company Endpoints provide access to comprehensive company information including profiles, executive compensation, company notes, employee data, shares float information, and earnings call transcripts.

<CodeBlock language="typescript">
  {`type CompanyTranscriptData = [number, number, string];`}
</CodeBlock>

**Note**: The `CompanyTranscriptData` type represents an array where:

- `[0]`: Year (number)
- `[1]`: Quarter (number, 1-4)
- `[2]`: Date (string, ISO format)

## Error Handling

All company endpoints return a standardized response format with error handling:

<CodeBlock language="typescript">
  {`interface UnwrappedAPIResponse<T> {
  success: boolean;
  data: T | null;
  error?: string;
}`}
</CodeBlock>

When an error occurs, the response will include:

- `success: false`
- `data: null`
- `error: string` - Description of the error

## Examples

### Complete Company Analysis

<CodeBlock language="typescript">
  {`// Get comprehensive company information
const [profile, compensation, employeeData, sharesFloat, transcriptData] = await Promise.all([
  fmp.company.getCompanyProfile('AAPL'),
  fmp.company.getExecutiveCompensation('AAPL'),
  fmp.company.getHistoricalEmployeeCount('AAPL'),
  fmp.company.getSharesFloat('AAPL'),
  fmp.company.getCompanyTranscriptData('AAPL')
]);

// Analyze company metrics
if (profile.success && profile.data) {
console.log('Company:', profile.data.companyName);
console.log('Market Cap:', profile.data.mktCap);
console.log('Industry:', profile.data.industry);
console.log('Employees:', profile.data.fullTimeEmployees);
}

// Check available transcripts
if (transcriptData.success && transcriptData.data) {
console.log('Available transcripts:', transcriptData.data.length);
const latestTranscript = transcriptData.data[0];
console.log('Latest transcript:', \`Q\${latestTranscript[1]} \${latestTranscript[0]} - \${latestTranscript[2]}\`);
}`}

</CodeBlock>

### Historical Analysis

<CodeBlock language="typescript">
  {`// Get historical data for analysis
const [historicalEmployees, historicalShares] = await Promise.all([
  fmp.company.getHistoricalEmployeeCount('AAPL'),
  fmp.company.getHistoricalSharesFloat('AAPL')
]);

// Analyze trends
if (historicalEmployees.success && historicalEmployees.data) {
const employeeGrowth = historicalEmployees.data
.sort((a, b) => new Date(b.filingDate).getTime() - new Date(a.filingDate).getTime())
.slice(0, 5);

console.log('Recent employee count trends:', employeeGrowth);
}`}

</CodeBlock>
