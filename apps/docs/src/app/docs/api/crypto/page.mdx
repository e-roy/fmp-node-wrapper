## Get Crypto Quote

Retrieve real-time cryptocurrency quote data including price, change, volume, and market cap.

<CodeBlock language="typescript">
  {`const quote = await fmp.crypto.getQuote({ symbol: 'BTCUSD' });`}
</CodeBlock>

## Get Historical Data

Retrieve historical cryptocurrency price data with flexible date ranges.

<CodeBlock language="typescript">
  {`const historicalData = await fmp.crypto.getHistoricalPrice({
  symbol: 'BTCUSD',
  from: '2024-01-01',
  to: '2024-01-31'
});`}
</CodeBlock>

<ParameterTable
  parameters={[
    {
      name: 'symbol',
      type: 'string',
      required: true,
      description: 'Crypto symbol',
    },
    {
      name: 'from',
      type: 'string',
      required: false,
      description: 'Start date (YYYY-MM-DD format)',
    },
    {
      name: 'to',
      type: 'string',
      required: false,
      description: 'End date (YYYY-MM-DD format)',
    },
    {
      name: 'timeseries',
      type: 'number',
      required: false,
      description: 'Number of days for timeseries',
    },
  ]}
/>

## Popular Cryptocurrencies

Here are some popular cryptocurrency symbols you can use:

- **BTCUSD** - Bitcoin / US Dollar
- **ETHUSD** - Ethereum / US Dollar
- **ADAUSD** - Cardano / US Dollar
- **DOTUSD** - Polkadot / US Dollar
- **LINKUSD** - Chainlink / US Dollar
- **LTCUSD** - Litecoin / US Dollar
- **XRPUSD** - Ripple / US Dollar
- **BCHUSD** - Bitcoin Cash / US Dollar

## Error Handling

Always check the success property before accessing data:

<CodeBlock language="typescript">
{`const quote = await fmp.crypto.getQuote({ symbol: 'INVALID' });

if (quote.success && quote.data) {
console.log('Crypto Price:', quote.data[0].price);
} else {
console.error('Error:', quote.error);
console.error('Status:', quote.status);
}`}

</CodeBlock>

## Rate Limiting

Cryptocurrency endpoints are subject to FMP's rate limits. For production applications, implement appropriate rate limiting and caching strategies.

## Market Data

Cryptocurrency markets operate 24/7, unlike traditional stock markets. This means you can get real-time data at any time of day.
