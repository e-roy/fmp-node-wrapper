# Stock Endpoints

The Stock Endpoints provide access to real-time and historical stock data, quotes, market capitalization, stock splits, and dividend history. For company profile information, see the [Company Endpoints](/docs/api/company).

## Get Stock Quote

Retrieve real-time stock quote data including price, change, volume, and market cap.

<CodeBlock language="typescript">

## Error Handling

Always check the success property before accessing data:

<CodeBlock language="typescript">
{`const quote = await fmp.stock.getQuote({ symbol: 'INVALID' });

if (quote.success && quote.data) {
console.log('Stock Price:', quote.data.price);
} else {
console.error('Error:', quote.error);
console.error('Status:', quote.status);
}`}

</CodeBlock>
