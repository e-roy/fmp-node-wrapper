# Stock Endpoints

The Stock Endpoints provide access to real-time and historical stock
data, quotes, and market information.

market cap.

## Get Historical Data

Retrieve historical price data with flexible date ranges and intervals.

<CodeBlock language="typescript">
  {`const historicalData = await fmp.getHistoricalPrice({
    symbol: 'AAPL',
    from: '2023-01-01',
    to: '2023-12-31',
    timeseries: 'daily'
});`}
</CodeBlock>

<ParameterTable
  parameters={[
    {
      name: 'symbol',
      type: 'string',
      required: true,
      description: 'Stock symbol',
    },
    {
      name: 'from',
      type: 'string',
      required: true,
      description: 'Start date (YYYY-MM-DD format)',
    },
    {
      name: 'to',
      type: 'string',
      required: true,
      description: 'End date (YYYY-MM-DD format)',
    },
    {
      name: 'timeseries',
      type: 'string',
      required: false,
      description:
        'Time interval: "1min", "5min", "15min", "30min", "1hour", "4hour", "daily" (default: "daily")',
    },
  ]}
/>

## Error Handling

Always check the success property before accessing data:

<CodeBlock language="typescript">
{`const quote = await fmp.getQuote({ symbol: 'INVALID' });

if (quote.success) {
    console.log('Stock Price:', quote.data[0].price);
} else {
    console.error('Error:', quote.error);
    console.error('Status:', quote.status);
}`}

</CodeBlock>

strategies.
