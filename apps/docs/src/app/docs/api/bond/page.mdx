# Bond Endpoints

Access bond data, treasury rates, and yield curve information.

## Overview

The bond endpoints provide access to bond market data, including treasury rates, bond prices, and yield curve information that are crucial for fixed income analysis.

## Quick Start

<CodeBlock language="typescript">
{`import { FMP } from 'fmp-node-api';

const fmp = new FMP({ apiKey: 'your-api-key' });

// Get treasury rates
const rates = await fmp.bond.getTreasuryRates();

// Get bond prices
const prices = await fmp.bond.getBondPrices();

// Get yield curve data
const yieldCurve = await fmp.bond.getYieldCurve();`}
</CodeBlock>

## Treasury Maturities

Available treasury maturities include:

- **1 Month** - Short-term rates
- **2 Month** - Short-term rates
- **3 Month** - Short-term rates (T-Bills)
- **6 Month** - Short-term rates (T-Bills)
- **1 Year** - Short-term rates (T-Bills)
- **2 Year** - Short-term rates (T-Notes)
- **3 Year** - Medium-term rates (T-Notes)
- **5 Year** - Medium-term rates (T-Notes)
- **7 Year** - Medium-term rates (T-Notes)
- **10 Year** - Long-term rates (T-Notes)
- **20 Year** - Long-term rates (T-Bonds)
- **30 Year** - Long-term rates (T-Bonds)

## Yield Curve Analysis

The yield curve is a key indicator for economic health:

<CodeBlock language="typescript">
{`// Analyze yield curve shape
const rates = await fmp.bond.getTreasuryRates();

if (rates.success) {
    const current = rates.data[0];
    const shortTerm = current.month3;  // 3-month rate
    const mediumTerm = current.year5;  // 5-year rate
    const longTerm = current.year10;   // 10-year rate
    
    // Normal yield curve (upward sloping)
    if (shortTerm < mediumTerm && mediumTerm < longTerm) {
      console.log('Normal Yield Curve - Healthy economy');
    }
    
    // Flat yield curve
    else if (Math.abs(shortTerm - longTerm) < 0.5) {
      console.log('Flat Yield Curve - Economic uncertainty');
    }
    
    // Inverted yield curve (recession indicator)
    else if (shortTerm > longTerm) {
      console.log('Inverted Yield Curve - Recession warning');
    }
}`}
</CodeBlock>

## Bond Spread Analysis

<CodeBlock language="typescript">
{`// Calculate credit spreads
const [treasury, corporate] = await Promise.all([
    fmp.bond.getTreasuryRates(),
    fmp.bond.getCorporateBonds()
]);

if (treasury.success && corporate.success) {
    const treasury10Y = treasury.data[0].year10;
    const corporate10Y = corporate.data.find(bond => 
      bond.maturity.includes('10')
    )?.yield;
    
    if (corporate10Y) {
      const spread = corporate10Y - treasury10Y;
      console.log('10-Year Credit Spread:', spread);
    }
}`}
</CodeBlock>

## Common Use Cases

### Fixed Income Dashboard

<CodeBlock language="typescript">
{`// Create fixed income dashboard
const [rates, prices, yieldCurve] = await Promise.all([
    fmp.bond.getTreasuryRates(),
    fmp.bond.getBondPrices(),
    fmp.bond.getYieldCurve()
]);

console.log('Current Rates:', rates.data[0]);
console.log('Bond Prices:', prices.data.slice(0, 5));
console.log('Yield Curve:', yieldCurve.data);`}
</CodeBlock>

### Duration Analysis

<CodeBlock language="typescript">
{`// Analyze interest rate sensitivity
const rates = await fmp.bond.getTreasuryRates();

if (rates.success) {
    const current = rates.data[0];
    const rateChange = current.year10 - current.year5;
    
    // Higher duration bonds are more sensitive to rate changes
    console.log('Rate Change (5Y to 10Y):', rateChange);
}`}
</CodeBlock>

## Market Hours

Bond markets typically trade:

- **Treasury Market** - 8:00 AM - 5:00 PM ET (Monday-Friday)
- **Corporate Bond Market** - 8:00 AM - 5:00 PM ET (Monday-Friday)
- **After-Hours** - Limited liquidity

## Key Bond Concepts

- **Yield** - Annual return on investment
- **Duration** - Interest rate sensitivity
- **Credit Spread** - Difference between corporate and treasury yields
- **Yield Curve** - Relationship between yields and maturities
- **Convexity** - Rate of change of duration
