# Financial Endpoints

The Financial Endpoints provide access to financial statements, ratios,
and fundamental data for companies.

## Get Income Statement

Retrieve income statement data including revenue, expenses, and profit
metrics.

<CodeBlock language="typescript">
  {`const incomeStatement = await fmp.getIncomeStatement({
    symbol: 'AAPL',
    period: 'annual',
    limit: 5,
});`}
</CodeBlock>

<ParameterTable
  parameters={[
    {
      name: 'symbol',

      description: 'Stock symbol',
    },
    {
      name: 'period',
      type: 'string',
      required: false,
      description: 'Period type: "annual" or "quarter" (default: "annual")',
    },
    {
      name: 'limit',
      type: 'number',
      required: false,
      description: 'Number of periods to retrieve (default: 5)',

## Error Handling

Always check the success property before accessing data:

<CodeBlock language="typescript">
{`const incomeStatement = await fmp.getIncomeStatement({ symbol: 'INVALID' });

if (incomeStatement.success) {
    console.log('Revenue:', incomeStatement.data[0].revenue);
} else {
    console.error('Error:', incomeStatement.error);
    console.error('Status:', incomeStatement.status);
}`}

</CodeBlock>

## Data Availability

Financial statement data availability depends on:

- **Company filing schedule** - Quarterly and annual reports
- **Data processing time** - FMP processes filings after SEC release
- **Subscription tier** - Some data may require premium plans
