# Mutual Fund Endpoints

The Mutual Fund Endpoints provide access to mutual fund holdings data. For mutual fund quotes, use the general quote endpoint documented in the [Quote Endpoints](/docs/api/quote).

## Available Methods

<ApiTable
  endpoints={[
    {
      method: 'GET',
      path: '/mutual-fund-holder/{symbol}',
      description: 'Get mutual fund holders of a stock',
    },
  ]}
/>

## Get Mutual Fund Holders

Retrieve the mutual funds that hold a specific stock and their positions.

<CodeBlock language="typescript">
  {`const holders = await fmp.mutualFund.getHolders({ symbol: 'AAPL' });`}
</CodeBlock>

<ParameterTable
  parameters={[
    {
      name: 'symbol',
      type: 'string',
      required: true,
      description: 'Stock symbol (e.g., "AAPL", "MSFT")',
    },
  ]}
/>

### Example Response

<CodeBlock language="typescript">
  {`{
    success: true,
    data: [
      {
        holder: 'Vanguard 500 Index Fund',
        shares: 1234567,
        dateReported: '2024-03-31',
        change: 10000,
        weightPercent: 12.34
      },
      {
        holder: 'Fidelity Contrafund',
        shares: 987654,
        dateReported: '2024-03-31',
        change: -5000,
        weightPercent: 9.87
      }
    ]
  }`}
</CodeBlock>

## Error Handling

Always check the `success` property before accessing data:

<CodeBlock language="typescript">
{`const holders = await fmp.mutualFund.getHolders({ symbol: 'INVALID' });

if (holders.success && holders.data) {
console.log('Mutual Fund Holders:', holders.data);
} else {
console.error('Error:', holders.error);
console.error('Status:', holders.status);
}`}

</CodeBlock>

## Notes

- The `getHolders` method returns the mutual funds that hold a given stock and their respective positions.
- For mutual fund quotes, use the [Quote Endpoints](/docs/api/quote).
- All mutual fund data is sourced from Financial Modeling Prep's comprehensive financial database.
