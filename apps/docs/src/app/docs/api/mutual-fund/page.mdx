# Mutual Fund Endpoints

Access mutual fund data including quotes, profiles, and performance information.

## Overview

The mutual fund endpoints provide access to mutual fund market data, including real-time quotes, fund profiles, performance metrics, and fund listings.

## Quick Start

<CodeBlock language="typescript">
{`import { FMP } from 'fmp-node-api';

const fmp = new FMP({ apiKey: 'your-api-key' });

// Get mutual fund quote
const quote = await fmp.mutualFund.getQuote({ symbol: 'VFINX' });

// Get mutual fund profile
const profile = await fmp.mutualFund.getProfile({ symbol: 'VFINX' });

// Get mutual fund list
const funds = await fmp.mutualFund.getMutualFundList();`}

</CodeBlock>

## Available Methods

### getQuote

Get real-time mutual fund quote.

<ApiTable
  endpoints={[
    {
      method: 'GET',
      path: '/quote/{symbol}',
      description: 'Get real-time mutual fund quote',
    },
  ]}
/>

**Parameters:**

| Parameter | Type     | Required | Description                 |
| --------- | -------- | -------- | --------------------------- |
| `symbol`  | `string` | Yes      | Fund symbol (e.g., 'VFINX') |

**Example:**

<CodeBlock language="typescript">
{`const quote = await fmp.mutualFund.getQuote({ symbol: 'VFINX' });

console.log('NAV:', quote.data[0].price);
console.log('Change:', quote.data[0].change);
console.log('Volume:', quote.data[0].volume);
}`}

</CodeBlock>

## Fund Categories

Mutual funds can be categorized by their investment focus:

- **Equity Funds** - Invest primarily in stocks
- **Bond Funds** - Invest primarily in bonds
- **Balanced Funds** - Mix of stocks and bonds
- **Index Funds** - Track specific market indices
- **Sector Funds** - Focus on specific industry sectors
- **International Funds** - Invest in foreign markets
- **Money Market Funds** - Short-term, low-risk investments

## Fund Analysis

### Expense Ratio Analysis

<CodeBlock language="typescript">
{`// Compare expense ratios
const [fund1, fund2] = await Promise.all([
    fmp.mutualFund.getProfile({ symbol: 'VFINX' }),
    fmp.mutualFund.getProfile({ symbol: 'FAGIX' })
]);

if (fund1.success && fund2.success) {
const expense1 = fund1.data[0].expenseRatio || 0;
const expense2 = fund2.data[0].expenseRatio || 0;

    console.log('VFINX Expense Ratio:', expense1);
    console.log('FAGIX Expense Ratio:', expense2);
    console.log('Difference:', expense2 - expense1);

}`}

</CodeBlock>

## Rate Limiting

Mutual fund endpoints are subject to FMP's rate limits. For production applications, consider implementing appropriate rate limiting strategies.

## NAV vs Market Price

Mutual funds trade at their Net Asset Value (NAV), which is calculated once per day after market close. This differs from ETFs, which trade throughout the day at market prices.

## Key Fund Metrics

- **NAV** - Net Asset Value per share
- **Expense Ratio** - Annual management fees
- **Load** - Sales charges (if any)
- **Turnover Ratio** - Trading activity level
- **Yield** - Income distribution rate
- **Beta** - Volatility relative to market
