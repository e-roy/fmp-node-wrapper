# Documentation

Welcome to the FMP Node Wrapper documentation. Choose the library that fits your needs:

## FMP Node API

The core API wrapper for direct access to the Financial Modeling Prep API.

- **Complete TypeScript Support** - Full type safety for all API responses
- **All FMP Endpoints** - Comprehensive coverage of the FMP API
- **Built-in Validation** - Input validation and error handling
- **Modular Design** - Import only what you need

<CodeBlock language="bash">{`npm install fmp-node-api`}</CodeBlock>

<CodeBlock language="typescript">
{`import { FMP } from 'fmp-node-api';

// Initialize with API key
const fmp = new FMP({ apiKey: 'your-api-key' });

// Get stock quote
const quote = await fmp.quote.getQuote('AAPL');`}

</CodeBlock>

[View API Documentation →](/docs/api)

## FMP Tools

AI tools for integrating FMP data with AI frameworks like Vercel AI SDK.

- **Vercel AI SDK Integration** - Ready-to-use tools for AI applications
- **Multi-platform Support** - Compatible with various AI frameworks
- **Easy Configuration** - Just set your API key and start building
- **Chatbot Ready** - Perfect for financial chatbots and assistants

<CodeBlock language="bash">{`npm install fmp-ai-tools ai @ai-sdk/openai`}</CodeBlock>

<CodeBlock language="typescript">
{`
import { openai } from '@ai-sdk/openai';
import { streamText, convertToModelMessages, stepCountIs } from 'ai';
import { fmpTools } from 'fmp-ai-tools/vercel-ai';

const result = streamText({
model: openai('gpt-4o-mini'),
messages: convertToModelMessages(messages),
tools: fmpTools,
stopWhen: stepCountIs(5),
});
`}

</CodeBlock>

[View Tools Documentation →](/docs/tools)

## Getting Started

1. **Choose your library** - API wrapper for direct access or AI tools for AI integration
2. **Install the package** - Use npm, yarn, or pnpm
3. **Set up your API key** - Get your FMP API key from [Financial Modeling Prep](https://site.financialmodelingprep.com/developer/docs/stable)
4. **Start building** - Follow the documentation for your chosen library
